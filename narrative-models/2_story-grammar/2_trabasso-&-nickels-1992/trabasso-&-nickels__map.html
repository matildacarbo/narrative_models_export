Trabasso & Nickels_Coding schema

The description of the coding schema is incomplete. This is a reconstruction based on their examples of transcripts (p.257, 263) and schemas (p.255, 256, 262), as well as their analysis (p.268-269).

Transcription was divided in:

  • Set of clauses narrating the same picture (p.257)
  • Subdivided in clauses (p.254, 257)
expression labelled labelling what labelling how (1): category complexity labelling how (2): categories used
clause episode ID = metadata Number (Nth episode)
clause picture ID = metadata Number (Nth picture)
clause clause ID = metadata Number (Nth clause)
clause macro unit + ID = metadata 6 categories + number S, E, IR, G, A, O. + Number (Nth unit of given type)
some clauses outcome subtype 3 categories O+, O-, Oo
clause character(s) involved story-specific; 4 categories in this case boy, boy & dog, dog, frog
clause level of goal hierarchy story-specific; 3 categories in this case level 1, level 2 (subgoal), level 3 (sub-sub-goal)
episode, and macro-units connections between & within episodes 4 categories enabling, motivational, psychological, physical

Labelled data was used to create causal network schemas.

Trabasso & Nickels_Terms

Few definitions, a lot of paraphrasing + deduction necessary. Some terms are hopefully(?!) defined in Trabasso et al 1989.
Abbreviations included because they directly connect terms & coding schema.

Context_external Context_internal Term Definition
macro_X episode "[Setting, event, internal response, goal, attempt and outcome] (...) These categories together constitute an episode." (p.252)
macro_X setting (S) No def. Presumably introduction of the time and place
macro_X event (E) An event that starts the narrative causal chain = paraphrase of p.252-253.
macro_X internal response; internal reaction (IR) Psychological effect of the Event, motivating the Goal; "[Setting + event] cause an internal reaction." (p.253)
macro_X core goal (G) Set of goal & subgoals (outcomes the protagonist seeks to achieve), motivated by Internal Response + motivating the Attempt = paraphrase of p.253
macro_X core attempt (A) Action taken to achieve Goal + resulting in Outcome.
macro_X core outcome (O) Situation after Attempt.
outcome successful outcome (O+) Outcome that matches Goal
outcome unsuccesful/failed outcome (O-) Outcome that does not match Goal
outcome neutral outcome (Oo) Outcome not involving success or failure with respect to the Goal (paraphrase of p.266)
macro_X class of unit GAO unit, core NEEDS UPDATE, minimum elements required for an episode
non-GAO unit "a clause or a set of clauses that maintain a topic but does not allow the inference of a goal" p.264

p.253 First paragraph explains that all units can enable units of the same kind, not just next unit in causal sequence. "Events can (...) cause other events, (...) Goals can motivate goals (...)"
